📌 Metode 1: Print dari Browser (Termudah)

  1. Buka presentasi HTML di browser (Chrome/Firefox/Safari/Edge)
  2. Tekan: Ctrl+P (Windows) atau Cmd+P (Mac)
  3. Di destination pilih: "Save as PDF"
  4. Klik Save dan pilih lokasi penyimpanan
  5. ✅ Selesai! File PDF siap digunakan
💡 Tips per Browser:
Chrome: Ctrl+P → More Settings → Margins: None → Save as PDF
Firefox: Ctrl+P → "Save to PDF" → Save
Safari: Cmd+P → PDF (dropdown) → Save as PDF
Edge: Ctrl+P → "Microsoft Print to PDF" → Print

📌 Metode 2: Akses Online (GitHub Pages)

Presentasi sudah di-deploy ke GitHub Pages:

https://deputisppti.github.io/websiteBPJS/PRESENTASI_WEBSITE_BPJS.html
  1. Buka URL di atas di browser
  2. Tekan: Ctrl+P atau Cmd+P
  3. Save as PDF
  4. ✅ File PDF tersimpan

📌 Metode 3: Jalankan Lokal dengan Server

  1. Install Node.js (jika belum): https://nodejs.org
  2. Buka Terminal/Command Prompt
  3. Masuk ke folder project:
    cd websiteBPJS
  4. Jalankan server:
    npx http-server -p 8080
  5. Buka browser: http://localhost:8080/PRESENTASI_WEBSITE_BPJS.html
  6. Tekan: Ctrl+P → Save as PDF
  7. ✅ PDF siap!
🐍 Alternatif: Gunakan Python 3
python -m http.server 8080

Kemudian buka: http://localhost:8080/PRESENTASI_WEBSITE_BPJS.html

🎯 Best Practices PDF Export

  1. Browser: Gunakan Chrome/Edge untuk hasil terbaik
  2. Margins: Set ke "None" atau "Minimal" untuk layout optimal
  3. Background: Enable "Background graphics" untuk warna BPJS sempurna
  4. Format: Ukuran paper "A4" portrait mode
  5. Scale: Pertahankan 100% scale asli
