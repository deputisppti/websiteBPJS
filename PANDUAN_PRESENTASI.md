# 📊 Panduan Presentasi Website Portal Layanan Digital BPJS Kesehatan

## 📁 File-File Presentasi yang Tersedia

Repository ini sudah dilengkapi dengan **3 format presentasi**:

### 1. **PRESENTASI_WEBSITE_BPJS.html** ⭐ (Recommended)
- **Format:** HTML Interaktif (bisa dibuka langsung di browser)
- **Fitur:** 
  - ✅ 14 slide lengkap
  - ✅ Navigasi dengan tombol Previous/Next
  - ✅ Keyboard navigation (← → arrow keys)
  - ✅ Progress bar real-time
  - ✅ Responsive design (mobile-friendly)
  - ✅ Design profesional dengan warna BPJS
- **Cara Akses:**
  - Buka file langsung di browser: [PRESENTASI_WEBSITE_BPJS.html](PRESENTASI_WEBSITE_BPJS.html)
  - Atau download lalu buka dengan double-click
- **Kelebihan:** Tidak perlu software khusus, bisa dibuka di browser apa saja

### 2. **PRESENTASI_WEBSITE_BPJS.md** 📝
- **Format:** Markdown (dokumentasi tertulis)
- **Isi:** Struktur lengkap 14 slide dengan deskripsi detail
- **Cara Menggunakan:**
  - Copy konten ke PowerPoint/Google Slides/Canva
  - Tambahkan screenshot dari website BPJS
  - Sesuaikan design dengan branding BPJS

### 3. **Website Asli** 🌐
- **File:** `index.html` (website BPJS Kesehatan)
- **Cara Menjalankan:**
  ```bash
  npx http-server -p 8080
  ```
  Lalu buka: `http://localhost:8080`

## 📱 Akses Website BPJS Saat Presentasi

Untuk demo interaktif, buka website asli di tab browser lain:

```bash
# Terminal/Command Prompt
npx http-server -p 8080

# Atau menggunakan Python 3
python -m http.server 8080

# Atau menggunakan Node.js
node -e "require('http').createServer((q,s)=>require('fs').createReadStream(require('path').join(__dirname,q.url.slice(1)||'index.html')).pipe(s)).listen(8080)"
```

Buka: `http://localhost:8080`

## ❓ FAQ

### **Q: Bagaimana cara membuka presentasi HTML?**
A: Buka file `PRESENTASI_WEBSITE_BPJS.html` dengan browser apa saja (Chrome, Firefox, Safari, Edge). Tidak perlu software khusus!

### **Q: Bisa di-share dengan peserta?**
A: Ya! Push ke GitHub Pages atau share link langsung. Peserta bisa buka di browser tanpa perlu download.

### **Q: Gimana caranya convert ke PowerPoint?**
A: Buka browser > Ctrl+P > Save as PDF > Import ke PowerPoint. Atau copy manual ke slide.

### **Q: Bisa customize slide?**
A: Ya! Edit file HTML langsung (ubah text, warna, icon) atau download markdown lalu paste ke PowerPoint.

### **Q: Apakah bisa offline?**
A: Ya! Presentasi HTML bekerja 100% offline tanpa koneksi internet.

### **Q: Gimana caranya print presentasi?**
A: Buka HTML di browser > Ctrl+P > Pilih printer/Save as PDF > Print/download sesuai kebutuhan.
